Object that contains the attribute key, its model and a list of events to
watch in order to keep its value synchronized.

## Example

```ts
const options = {
  name: 'name',
  model: user,
  watchEvents: ['rename', 'change:name'],
} as UseModelAttributeOptions<UserAttributes, 'name'>;
```
